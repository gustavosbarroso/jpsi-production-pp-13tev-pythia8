# jpsi-production-pp-13tev-pythia8
# J/ψ Production in pp Collisions at 13 TeV

This project studies the production of J/ψ mesons in proton-proton collisions at √s = 13 TeV using Monte Carlo simulations with Pythia8.

## Requirements

- Pythia8
- ROOT
- C++ compiler

## Description

The code generates pp collision events and analyzes the transverse momentum distribution of J/ψ particles.
